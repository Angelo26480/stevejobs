Struttura del Documento:

  1. <!DOCTYPE html>: Definisce il tipo di documento HTML5.
  2. <html>: Rappresenta il radice di un documento HTML.
  3. <head>: Contiene metadati (come il titolo, collegamenti a fogli di stile e script) relativi al documento HTML.
    • <title>: Definisce il titolo del documento HTML.
    • <meta>: Fornisce metadati sul documento HTML.
    • <link>: Collega il documento HTML a un foglio di stile esterno o a altre risorse.
    • <style>: Definisce gli stili CSS all'interno del documento HTML.
    • <script>: Incorpora script o collegamenti a script esterni nel documento HTML.
    • <base>: Specifica l'URL base per risorse relative in un documento HTML.
  4. <body>: Rappresenta il contenuto principale del documento HTML.

Intestazioni e Paragrafi:

  1. <header>: Rappresenta l'intestazione di una sezione o di un documento.
  2. <footer>: Rappresenta il piè di pagina di una sezione o di un documento.
  3. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Definiscono i livelli di intestazione del testo.
  4. <p>: Rappresenta un paragrafo di testo.
  5. <hr>: Crea una riga orizzontale per separare il contenuto.

Testo e Formattazione:

  1. <strong>: Rende il testo in grassetto.
  2. <em>: Rende il testo in corsivo.
  3. <mark>: Evidenzia il testo.
  4. <del>: Indica il testo eliminato.
  5. <ins>: Indica il testo aggiunto.
  6. <sub>: Rende il testo come pedice.
  7. <sup>: Rende il testo come apice.
  8. <abbr>: Definisce un'abbreviazione.
  9. <blockquote>: Rappresenta una citazione lunga.
  10. <q>: Rappresenta una citazione breve.
  11. <cite>: Definisce il titolo di una citazione o di un lavoro.

Liste:

  1. <ul>: Rappresenta un elenco non ordinato.
  2. <ol>: Rappresenta un elenco ordinato.
  3. <li>: Rappresenta un elemento di lista all'interno di un elenco.
  4. <dl>: Rappresenta una lista di definizioni.
  5. <dt>: Definisce il termine nella lista di definizioni.
  6. <dd>: Fornisce la definizione nel <dl>.

Tabelle:

  1. <table>: Rappresenta una tabella.
  2. <caption>: Fornisce una descrizione o un titolo per una tabella.
  3. <thead>, <tbody>, <tfoot>: Definiscono l'intestazione, il corpo e il piè di pagina di una tabella, rispettivamente.
  4. <tr>: Rappresenta una riga in una tabella.
  5. <th>: Rappresenta una cella di intestazione di una tabella.
  6. <td>: Rappresenta una cella di dati in una tabella.

Form:

  1. <form>: Rappresenta un modulo HTML.
  2. <input> : Definisce un campo di input per il modulo.
  3. <button>: Rappresenta un pulsante cliccabile.
  4. <select>: Rappresenta un menu a discesa.
  5. <option>: Definisce un'opzione in un menu a discesa.
  6. <textarea>: Definisce un campo di testo multiriga.
  7. <label>: Definisce l'etichetta di un campo di input.
  8. <fieldset>: Rappresenta un gruppo di controlli del modulo associati.
  9. <legend>: Fornisce una legenda per il <fieldset>.

Multimedia:

  1. <img>: Incorpora un'immagine nel documento HTML.
  2. <audio>: Incorpora un file audio nel documento HTML.
  3. <video>: Incorpora un video nel documento HTML.
  4. <source>: Specifica una risorsa multimediale per <audio> e <video>.
  5. <track>: Definisce una traccia di testo per <video> e <audio>.

Link e Ancore:

  1. <a>: Definisce un collegamento ipertestuale.
  2. <nav>: Definisce una sezione di navigazione.
  3. <menu>: Definisce un menu di navigazione.

Sezioni:

  1. <section>: Rappresenta una sezione del documento.
  2. <article>: Rappresenta un contenuto autonomo e potenzialmente riciclabile.
  3. <aside>: Rappresenta il contenuto correlato, spesso laterale, come le barre laterali in un layout.
  4. <main>: Rappresenta il contenuto principale o centrale del documento.

Contenuto Incorporato:

  1. <iframe>: Incorpora un'altra pagina HTML in una pagina HTML.
  2. <embed>: Incorpora contenuti esterni come plugin multimediale.
  3. <object>: Incorpora oggetti esterni, come file multimediale o applet Java.

Progresso e Controllo:

  1. <progress>: Rappresenta lo stato di avanzamento di un'attività.
  2. <meter>: Rappresenta uno specifico valore in un intervallo.

Struttura Logica:

  1. <header>: Fornisce un'intestazione per una sezione o per l'intero documento.
  2. <footer>: Fornisce un piè di pagina per una sezione o per l'intero documento.
  3. <nav>: Rappresenta un insieme di link di navigazione.
  4. <div>: Rappresenta una divisione logica o un contenitore generico.
  5. <span>: Rappresenta un testo o un contenuto in linea.
  6. <br>: Crea un'interruzione di linea.

Testo e Caratteri:

  1. <ruby>, <rt>, <rp>: Rappresenta una notazione ruby per l'annotazione del testo.
  2. <bdo>: Sovrascrive la direzione del testo.

Altri:

  1. <canvas>: Fornisce uno spazio riservato per il disegno dinamico di grafica.
  2. <details>: Rappresenta un controllo che permette di visualizzare o nascondere ulteriori dettagli.
  3. <summary>: Fornisce un'etichetta o una legenda per l'elemento <details>.
  4. <dialog>: Rappresenta una finestra di dialogo o una finestra popup.
  5. <slot>: Fornisce una destinazione per un'istanza di un template.
