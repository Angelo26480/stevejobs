Lista dettagliata di alcuni comandi CSS e come utilizzarli

1. Selettori:

  • element: Seleziona tutti gli elementi di tipo specificato. Ad esempio, p selezionerà tutti i paragrafi.
  • .classe: Seleziona tutti gli elementi con la classe specificata. Ad esempio, .intro selezionerà tutti gli elementi con la classe "intro".
  • #id: Seleziona l'elemento con l'id specificato. Ad esempio, #header selezionerà l'elemento con l'id "header".
  • [attributo]: Seleziona gli elementi con l'attributo specificato. Ad esempio, [type] selezionerà tutti gli elementi che hanno un attributo "type".
  • element, element: Seleziona più elementi separati da virgola. Ad esempio, h1, h2, h3 selezionerà tutti gli elementi di intestazione h1, h2 e h3.
  • element > element: Seleziona un elemento figlio diretto. Ad esempio, div > p selezionerà tutti i paragrafi che sono figli diretti di un elemento div.
  • element + element: Seleziona un elemento fratello immediatamente successivo. Ad esempio, h2 + p selezionerà il primo paragrafo che segue immediatamente un elemento di intestazione h2.
  • element ~ element: Seleziona tutti gli elementi fratelli successivi. Ad esempio, p ~ span selezionerà tutti gli elementi span che sono fratelli successivi di un paragrafo.
  • Pseudoclassi di Stato: Questi selettori permettono di selezionare elementi in base al loro stato, come ad esempio :hover per quando il mouse passa sopra un elemento, :active per quando un elemento è attivato, o :focus per quando un elemento ha lo stato di "focus".
  • Pseudoelementi: Consentono di selezionare parti specifiche di un elemento, come ad esempio ::before per aggiungere contenuto prima dell'elemento selezionato o ::after per aggiungere contenuto dopo l'elemento selezionato.
  • Selettori di Figlio Nth-child: Consentono di selezionare gli elementi in base alla loro posizione all'interno di un genitore, come ad esempio `:nth-child(n)` per selezionare tutti gli elementi figli che sono l'n-esimo figlio del loro genitore.
  • Selettori di Figlio Nth-of-type: Simili ai selettori di Figlio Nth-child, ma si concentrano sul tipo di elemento anziché sulla posizione.
  • Selettori di Attributo Avanzati: Oltre ai semplici selettori di attributo, ci sono anche quelli più complessi come ^= (inizia con), $= (finisce con), *= (contiene), che consentono di selezionare gli elementi in base a determinati pattern all'interno degli attributi.
  • Selettori di Testo e Contenuto: Per esempio, :contains(text) per selezionare elementi che contengono un certo testo, o :empty per selezionare elementi senza figli.
  • Selettori di Stato di Valore di Input: Per esempio, :checked per selezionare input radio o checkbox che sono stati selezionati.

2. Proprietà di layout:

  • display: Specifica il tipo di display per un elemento. Ad esempio, display: block renderà un elemento come blocco. Oppure il comando display: flex crea un Flex Container l'elemento a cui viene applicata la regola display: flex; diventa un "flex container". Questo significa che gli elementi figlio diretti di questo elemento verranno trattati come "flex items".
    • Imposta la Direzione Predefinita: Per impostazione predefinita, un flex container organizza i suoi flex items in una riga (da sinistra a destra) se l'attributo flex-direction non viene specificato. Puoi modificare la direzione del layout impostando la proprietà flex-direction sul flex container.
    • Imposta la Dimensione dei Flex Items: Gli elementi figlio diretti dell'elemento con display: flex; vengono considerati "flex items" e verranno posizionati all'interno del flex container. La dimensione dei flex items può essere controllata utilizzando le proprietà CSS come flex-grow, flex-shrink e flex-basis.
    • Allinea i Flex Items: Puoi controllare l'allineamento degli elementi figlio diretti del flex container utilizzando le proprietà CSS come justify-content per l'asse principale e align-items per l'asse trasversale.
  • float: Posiziona un elemento a sinistra o a destra del contenuto. Ad esempio, float: left sposterà un elemento a sinistra.
  • position: Specifica il tipo di posizionamento per un elemento. Ad esempio, position: relative posizionerà un elemento in relazione alla sua posizione originale.
  • top, right, bottom, left: Specifica la posizione di un elemento posizionato. Ad esempio, top: 50px sposterà un elemento di 50 pixel dall'alto.
  • z-index: Specifica lo stacking order degli elementi posizionati. Ad esempio, z-index: 1 metterà un elemento sopra gli altri con uno z-index inferiore.

3. Dimensioni e spaziatura:

  • width, min-width, max-width: Specifica la larghezza di un elemento.
  • height, min-height, max-height: Specifica l'altezza di un elemento.
  • margin: Specifica il margine esterno di un elemento.
  • padding: Specifica il padding interno di un elemento. Ovvero quanto distante dal bordo interno un elemento deve trovarsi
  • border: Specifica il bordo di un elemento.

4. Colori e sfondi:

  • color: Specifica il colore del testo.
  • background-color: Specifica il colore di sfondo di un elemento.
  • opacity: Specifica l'opacità di un elemento.

5. Testo:

  • font-family: Specifica il tipo di carattere del testo.
  • font-size: Specifica la dimensione del carattere del testo.
  • font-weight: Specifica lo spessore del carattere del testo.
  • text-align: Specifica l'allineamento del testo.
  • text-decoration: Specifica la decorazione del testo (sottolineato, barrato, etc.).

6. Effetti visivi:

  • box-shadow: Aggiunge un'ombra a un elemento.
  • transition: Specifica una transizione per le proprietà CSS.
  • transform: Applica una trasformazione 2D o 3D a un elemento.
  • animation: Specifica un'animazione per un elemento.

7. Pseudoclassi e pseudoelementi:

  • :hover, :active, :focus: Pseudoclassi per gli stati di interazione.
  • ::before, ::after: Pseudoelementi per aggiungere contenuto prima o dopo un elemento.

Breve guida su come implementare una formattazione CSS

Per creare una pagina CSS esistono vari metodi, un modo "semplice" è, per esempio, quello di creare innanzitutto la pagina HTML, definendo quindi i vari tag ai testi e utilizzare poi il tag <div > alle varie sezioni interessate.

Il tag <div >viene utilizzato principalmente per raggruppare e stilizzare altri elementi HTML fornendo una struttura e una formattazione alla pagina (in poche parole raggrupperà i vari elementi della pagina in "blocchi"). All'interno del tag div è poi possibile specificare anche un id o una classe, che renderà più semplice l'applicazione degli elementi css alla pagina richiamando il nome dell'id o della classe.

Per fare un esempio, se si ha un testo con menù e si vuole posizionare il menù a sinistra e il testo a destra, si deve innanzitutto racchiudere l'intero body del testo html in un unico blocco usando il tag <div> ,in modo da creare una base per l'applicazione degli stili CSS.

Dopo di ciò si racchiudono, con due tag <div > separati tra loro, sia il menù che il testo in modo da avere il menù e il testo in due blocchi separati. Ai tre div creati verranno quindi implementati delle classi/id come ad esempio "documento", menù e testo :

    <div class = "documento">

questo servirà ,come detto sopra, a identificar meglio ad identificare meglio i blocchi per la formattazione css.

Fatto ciò verrà aperta la pagina html, appena creata, sul browser cliccando sul file o aprendolo direttamente da visual studio.

La pagina non mostrerà nulla di particolare, questo perchè il tag <div > non applica cambiamenti visibili. Per poter modificare e abbellire la pagina usando il CSS, bisogna cliccare col tasto destro sulla pagina e poi su analizza (oppure F12), questo aprirà una finestra per poter agire direttamente sulla pagina html. Andare infine sulla finestra {}Editor Stili (se si è su firefox).

In questa sezione sarà possibile implementare a piacimento i vari comandi CSS vedendo direttamente la modifiche alla pagina, per richiamare quindi i vari blocchi da modificare si utilizzeranno gli id o le classi dei vari <div >(per gli ID si utilizzerà il # per le classi il .)

Tornando quindi all'esempio di sopra, del menù a sinistra, è utile utilizzare il comando "display : flex" in modo da "trasformare" il blocco div in blocco flex (rendendoli quindi più "flessibili" all'interno della pagina). Ad esempio : 

            .documento {
              display: flex;
            }
            .navigazione {
              display: flex;
              flex-direction: column;
            }

con il comando flex-direction: column; il menù verrà posizionato a sinistra

Sarà poi possibile introdurre altri comandi per aggiundere altri stili/abbellimenti (Non aggiornare la pagina se non si vogliono perdere le varie modifiche)

Quanto fatto fino ad ora però non implementa realmente il codice css alla pagina, serviva solo per vedere come i comandi css avrebbero modificato visivamente la pagina.

Per poter implementare le modifiche, bisogna creare un file nome.css (o style.css) su visual studio, copiare e incollare i vari comandi creati sulla pagina e salvare. Infine sull'head della nostra pagina HTML scrivere:

    <link rel="stylesheet" href="nome.css" />

Questo permetterà di linkare la pagina html al file css appena creato.
